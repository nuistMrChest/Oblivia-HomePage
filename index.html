OBLIVIA

a lightweight script language for calling other executable files

{
    let a=100;
    while a>0:{
        print a;
        print "\n";
        if a%2==0:{
            print "AAAAA\n";
        }
        else{
            print "BBBBB\n";
        }
        if a==50:{
            break;
        }
        a=a-1;
    }
}

Simple grammer


A statement based grammer similar to c programming language, but also grabed idea form other languages like python

Fully open source


All the source code are available on github(including the bugs) under MIT license

Easy to build


If the binary of your platform is not available now, don't worry. The source code uses nothing else but libc and stl, and we've provided CmakeLists.txt, so feel free to build it.

Future

This project is still in early stages. Its already functioning as a programing language, but the core feature, which is to call other executables just like calling functions, is still not implemented yet. You are welcome to leave your ideas or bug report to zookeeper.uob@outlook.com.
